# 💄 Beauty Website

A modern, responsive Beauty & Skincare website built using **HTML, CSS, and JavaScript**.  
This project is designed to showcase beauty products, services, and brand identity with a clean UI and smooth user experience.

---

## ✨ Features

- 🌸 Modern and elegant UI design  
- 📱 Fully responsive (mobile, tablet, desktop)  
- 🛍️ Product showcase section  
- ⭐ Testimonials section  
- 📞 Contact form section  
- 🎨 Smooth animations and hover effects  
- ⚡ Fast loading and lightweight structure  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox & Grid)  
- JavaScript (Vanilla JS)  
- Font Awesome Icons
